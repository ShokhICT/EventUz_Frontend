## Project Overview

EventHub Uzbekistan is a web-based platform designed to simplify event management. The system enables users to discover and register for events while providing administrators with tools to manage events, attendees, and event-related information through a centralized dashboard.

---

## Technology Stack

### Frontend

* Nuxt 3
* Vue 3
* Tailwind CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose

### Authentication

* JSON Web Tokens (JWT)

---

## Core Features

### User Features

* User Registration and Authentication
* Browse Upcoming Events
* Search and Filter Events
* Event Registration
* Profile Management
* My Events Dashboard

### Administrator Features

* Secure Admin Dashboard
* Event Management (CRUD)
* Category Management
* User Management
* Registration Management
* Reports and Statistics

## Project Goal

To develop a centralized event management platform that automates event organization processes, improves attendee management, and provides a better user experience for both organizers and participants.

---

## Objectives

* Develop a modern event management web application.
* Automate event registration and attendee management processes.
* Provide administrators with efficient event management tools.
* Implement a secure and scalable system using modern web technologies.
